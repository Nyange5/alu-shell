0-hello_world
a script that prints “Hello, World”, followed by a new line to the standard output.

a script that displays a confused smiley "(Ôo)'.

2-hellofile
Display the content of the /etc/passwd file.

3-twofiles
Display the content of /etc/passwd and /etc/hosts

4-lastlines
Display the last 10 lines of /etc/passwd

5-firstlines
Display the first 10 lines of /etc/passwd

6-third_line
Write a script that displays the third line of the file iacta.
The file iacta will be in the working directory
You’re not allowed to use sed

7-file
Write a shell script that creates a file named exactly \*\\'"Best School"\'\\*$\?\*\*\*\*\*:) containing the text Best School ending by a new line.

8-cwd_state
Write a script that writes into the file ls_cwd_content the result of the command ls -la. If the file ls_cwd_content already exists, it should be overwritten. If the file ls_cwd_content does not exist, create it.

9-duplicate_last_line

Duplicate (append) the last line of the file iacta to the end of the same file.

10-no_more_js

Delete all regular files ending with .js in the current directory and all subdirectories (directories themselves must not be deleted).

11-directories

Count all directories and sub-directories in the current directory.
Exclude . and .., include hidden directories.

12-newest_files

Display the 10 newest files in the current directory,
one per line, sorted from newest to oldest.

13-unique

Read words from standard input and print only the words that appear exactly once, sorted alphabetically.

14-findthatword

Display all lines containing “root” from /etc/passwd.

15-countthatword

Print the number of lines in /etc/passwd that contain the word “bin”.

16-whatsnext

Display lines containing “root” from /etc/passwd plus the 3 lines after each match.

17-hidethisword

Display all lines that do NOT contain “bin” from /etc/passwd.

18-letteronly

Display all lines from /etc/ssh/sshd_config that start with a letter (uppercase or lowercase).

19-AZ

Replace:

A → Z

c → e
from standard input.

20-hiago

Remove all letters c and C from standard input.

21-reverse

Reverse the input text from standard input.

22-users_and_homes

Display username and home directory from /etc/passwd,
formatted as user:home, sorted by username.

23-empty_casks

List empty files and empty directories (including hidden ones) in the current directory and subdirectories.
Print only names, one per line.

24-gifs

List all .gif regular files (including hidden ones) recursively.
Print file names without .gif, sorted case-insensitively by byte value.

25-acrostic

Decode an acrostic by printing the first letter of each input line, followed by a newline.

26-the_biggest_fan

From TSV web logs, display the 11 hosts/IPs with the most requests, ordered from most to least.
